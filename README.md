# Golang で、湯婆婆を実装する

## Usage:

```sh
go run src/main/main.go
```

or

```sh
go build src/main/main.go
./main
```

## Librarys:

* [fmt](https://golang.org/pkg/fmt/)
* [strings](https://golang.org/pkg/strings/#Split)
* [math/rand](https://golang.org/pkg/math/rand/)
